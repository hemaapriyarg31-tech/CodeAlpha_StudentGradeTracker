# Student Grade Tracker (Java)

This project is a console-based Java application developed as part of the CodeAlpha Internship.

## Features
- Add student details (name & marks)
- Calculate average, highest, and lowest marks
- Display student report
- Grade and pass/fail system
- Search student by name

## Technologies Used
- Java
- ArrayList
- Object-Oriented Programming (OOP)

## How to Run
1. Compile the program:
   javac StudentGradeTracker.java
2. Run the program:
   java StudentGradeTracker
